Data Lake and data processing

Within the Data Lake python will be used as the coding syntax for processing and data transformation within a utility called Apache Spark (specificly pyspark). All of the skills learned above are applicable to spark with minor exceptions which will be noted within the Saprk Documentation. Spark uses a concept called a Resilient Distributed Dataset or RDD to hold data and operate on it. The Spark utility contains a number of usefull libraries and modules including Spark SQL (SQL on RDD), Machine Learning, and more.

Once you are fimiliar with the base concepts above please read the Spark documentation located here. Note: The verson of Apache Spark that will be used is 2.0.0, this link is to the correct version's documentation.
